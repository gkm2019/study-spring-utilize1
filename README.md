# microsite-with-spring-boot
spring기반 microsite 서비스 개발을 위한 프로젝트 입니다.

* Self-Describtive Message와 HATEOAS를 만족하는 REST API를 이해합니다.
* 스프링 HATEOAS와 스프링 REST Docs 프로젝트를 활용할 수 있습니다.
* 테스트 주도 개발(TDD)에 익숙해 집니다.

## 프로젝트 정보
* windows 10
* Spring boot 2.3.4
* JDK 11
* H2, MySQL

## 의존성
* Web
* JPA
* HATEOAS
* TEST Docs
* H2
* MySQL
